Atualização de sistema!

Agora no histórico anual de faturamento é possível analisar os últimos 12 meses de produção dos itens de determinado cliente.

Todo este processo pode ser realizado de forma dinâmica, alterando a data para qualquer período o sistema sempre exibirá as informações com base nos últimos doze meses, tendo como base a data selecionada.

O sistema conta com a possibilidade de exportação dos dados em Excel para manipulação externa, as melhorias acima aplicam-se também ao histórico anual de entradas.
